Capability boundary

只有 HTML，
可以做到什麼？

HTML 可以描述內容

標題、段落、圖片、連結、表格、表單欄位、影片與清單。

只有 HTML 還無法完成

驗證帳密、儲存資料、查詢訂單、計算金額、判斷會員資格。

HTML 負責描述畫面內容；完整的 Web Application 還需要其他技術。
